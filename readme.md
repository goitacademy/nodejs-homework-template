General info:
In this project we use REST API to work with a set of contacts. User authentication/authorization logic was used here by using JWT.
Technologies applied:

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (JSON Web Tokens)
- Bcrypt
- Joi
- dotenv

To setup this app use the following command:
npm run start
